Tools for Information Literacy:  Spring 2018

Tools for Information is a general overview course of common computer architectures and the systems that enable them.

Class Objectives:

  1. Basic Understanding of Linux, Command Line, and Networks
    • Clients
    • Servers
    • Networks
    • GitHub
    • Basic Linux Commands
  2. Web Development
    • HTML and CSS fundamentals (as seen in this website)
  3. Document Markup
    • Document generation tools
  4. Spreadsheets
    • Microsoft Excel spreadsheets
  5. Databases
    • Micrososft access

This class has encouraged me to pursue a minor in Information Science at UNC, which will aid with my knowledge of Computer Science by offering me a new perspective as to how to use technology in applicable, every day circumstances. I'm excited to complete this course and to continue with my studies in information science.
